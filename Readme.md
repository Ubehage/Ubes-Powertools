# Ubes PowerTools

This is a collection of small tools and utilities, designed to test system performance.
Inspired by the old Windows 2000 Resource Kit.

---

This repo contains a collection of executables for stress-testing and exploring Windows system limits. 
Source code is maintained in separate repositories; this repo only contains ready-to-run binaries.


## Vision
- Empower the user to destroy their own pc, with a safety-switch at hand.

## Included products
- MemEater
  - Allocates large amounts of memory to test system stability under pressure.
- ShutdownTimer
  - A simple program for shutting down the computer after a specified amount of time.
- CPU-Stress
  - Can overload the CPU at will.
  - Per-core control. The user can overload certain cores, or all at once.
  
## Safety notice
- These programs all have the ability to crash Windows or potentially cause heat damage.
- Always save your work before testing.
- Use at your own risk.
